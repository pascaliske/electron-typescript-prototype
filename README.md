## Usage

### Development

To compile and start the app in development mode run:

```bash
yarn run start
```

### Packaging

To compile and package the app in production mode run:

```bash
yarn run package
```

### Linting

The following command checks the TypeScript code for linting errors:

```bash
yarn run lint
```
